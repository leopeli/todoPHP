# todoPHP

Adicionar novos item a lista;

Editar itens não concluídos;

Concluir itens ainda não concluídos;

Voltar itens concluídos para a lista;

Filtrar itens ativos;

Filtrar itens concluídos;

Limpar a lista de itens;

Contador de itens;


O que precisa ser feito
Criar uma única página onde seja possível visualizar todas as 
tarefas e executar todas as funcionalidades descritas.

As tarefas devem ser salvas em um banco de dados (MySQL).

Os filtros devem ser aplicados utilizando JavaScript.
Animações devem ser feitas utilizando
principalmente CSS, mas pode ser utilizado JavaScript também.

Pode ser utilizado qualquer framework PHP (preferencialmente CodeIgniter).
